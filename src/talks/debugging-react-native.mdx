### debugger-ui

- Developer Menu にて `Debug JS Remotely` を選択する
- ios は `⌘ | Ctrl + D` android は `⌘ | Ctrl + M`
- 公式のDebugger
- chromeのDeveloper toolsから開く

<Layout>

### react-devtools

- https://github.com/facebook/react-devtools/tree/master/packages/react-devtools
- chrome拡張の React Developer Tools の Standalone版
- Remote Debug JS が On になっていなくても使える:thinking_face:
- Profiler が使える
- 早い

</Layout>
